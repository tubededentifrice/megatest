# Megatest Instructions

Project-specific instructions for the /megatest skill.
This file is automatically maintained — edit freely, and the skill will also update it when it learns something new.

## Project Quirks

- This project is a CLI tool, not a web app. The `.megatest/` config exists to provide a demo report for the serve UI.
- The demo report is generated by `scripts/generate-test-report.cjs` — run it with `NODE_PATH=cli/node_modules node scripts/generate-test-report.cjs` to regenerate.
- The report uses a fixed commit hash `demo-report` so it's always stable.
